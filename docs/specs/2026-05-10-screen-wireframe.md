## 화면 흐름

```mermaid
flowchart TD
    A[메인/랜딩] --> E[AI 채팅 상담]
    A --> C[로그인]
    C --> E
    C --> F[마이페이지]
    F --> E
```

## 화면별 와이어프레임

### 1. 메인/랜딩 (`/`)

```
┌────────────────────────────────────┐
│  [로고]                 [로그인]      │
├────────────────────────────────────┤
│                                    │
│   보험 약관, 내 편에서 해석해드려요        │
│   청구 가능 여부를 미리 확인하세요         │
│                                    │
│         [지금 시작하기]               │
│                                    │
└────────────────────────────────────┘
```
